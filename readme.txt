Load Hotjar and prevent it from tracking admins, logged-in users, and IP addresses.

== Description ==
This plugin allows you to selectively load Hotjar in the header of your site. You can choose to prevent Hotjar from being loaded for admins, logged in users, and specific IP addresses.

Want to contribute? Check out the [GitHub repository](https://github.com/jxxe/advanced-hotjar).

== Installation ==
Once you've activated the plugin, enter your Hotjar Site ID and configure the plugin.
